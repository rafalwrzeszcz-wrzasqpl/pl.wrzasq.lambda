Git repository

All project sources and related resources are stored in Git repository. To standardize workflow with repository, WrzasqPl-Lambda follows Git-flow policy. In order to synchronize your repository you should call git flow init in your repository directory with following settings:

  • Branch name for production releases: - master
  • Branch name for “next release” development: - develop
  • Feature branches? - feature/
  • Release branches? - release/
  • Hotfix branches? - hotfix/
  • Support branches? - support/
  • Version tag prefix? - release-
